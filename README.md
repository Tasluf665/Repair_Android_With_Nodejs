# Repair - Electronic Device Repair Service App

Welcome to the documentation for the Repair app, an Android application built with React Native Expo. Repair is an e-commerce platform where users can hire technicians to repair their electronic devices. This project consists of three key components: the Android app (developed with React Native Expo), the backend server (powered by Node.js), and the admin panel (created with React). MongoDB serves as the primary database.

## Features

### Android App:

1. **User Authentication**:
   - Users can securely login using their email and password.
   - Google login integration for user convenience.

2. **User-Friendly Interface**:
   - An intuitive and user-friendly interface enhances the user experience.

3. **Payment System with SSLCOMMERZ**:
   - Seamless integration with the SSLCOMMERZ payment gateway for secure payment processing.

4. **Navigation**:
   - Smooth navigation between app pages for easy access to different features.

5. **Authentication**:
   - Role-based authentication ensures access control and security.
   
6. **Push Notifications**:
   - Real-time push notifications keep users informed and engaged.

### [Backend Server](https://github.com/Tasluf665/Backend_Repair)

1. **Login with Password**:
   - Secure login functionality using username and password.

2. **Google Login**:
   - Google-based authentication for a seamless login experience.

3. **Sending Notifications**:
   - Real-time notifications sent to the Repair Android app using the Expo server.

4. **Password Encryption**:
   - Robust password encryption to safeguard user data.

5. **Admin and Role-Based Authentication**:
   - Supports multiple user roles, including Admin, with role-based access control.

6. **Payment System with SSLCOMMERZ**:
   - Integration with the SSLCOMMERZ payment gateway for secure payment processing.

7. **Remote MongoDB Cloud Database**:
   - Connection to a remote MongoDB Atlas database for cloud-based data storage.


### [Admin Panel](https://github.com/Tasluf665/Repaire_Dashboard_Nodejs)

1. **Role-Based Authentication**:
   - Secure role-based authentication for administrators.

2. **Customer Orders Handling**:
   - Efficient management of customer orders.

3. **Agent and Technician Handling**:
   - Interactive handling of agents and technicians.

4. **Interactive Admin Panel**:
   - An intuitive and interactive admin panel for efficient management.

5. **Login with Password or Google Login**:
   - Multiple login options for administrators.

## Getting Started

To use the Repair app, follow these steps:

1. **Download Expo Go App**: Install the Expo Go app from the Google Play Store.

2. **Access the App**: Visit the following URL using the Expo Go app: [Repair Expo App](https://expo.dev/@tasluf/repair-nodejs?serviceType=classic&distribution=expo-go)

3. **Scan the QR Code**: Use the Expo Go app to scan the QR code provided.

4. **Wait for Installation**: Wait for a minute as the app is loaded.

5. **Ready to Go**: You're now ready to use the Repair app and explore its features.
